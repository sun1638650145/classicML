# classicML 
Python简单易用的经典机器学习框架
## 重要信息

你可以使用pip安装

```shell
pip install classicML
```

## version v0.1
1. 添加决策树
2. 决策树支持离散值、连续值
3. 基于信息熵、信息增益、基尼指数划分；支持预剪枝和后剪枝；暂不支持多变量决策树和缺失值处理（建议在读入数据集之前处理）

## version v0.2
1. 添加神经网络
2. 神经网络支持交叉熵损失函数和均方误差损失函数；支持的优化器有GradientDescent、SGD、Adam
